Количество участвующих в турнире команд неограничено.

Турнир проводится в две стадии: сперва групповой этап, затем плей-офф.

По итогам группового этапа в плей-офф проходят 1, 2 и 3 места каждой группы.

Количество команд в одной группе не может превышать 4 команды. Группы всегда формируются исходя из
принципа рандомного распределения, чтобы обеспечить максимально честные условия для всех команд.

Распределение мест в турнирной таблице группы происходит в соответствии с суммой набранных очков в каждом
матче:

• За победу команда получает три очка;

• За поражение команда получает ноль очков.

• За ничью команда получает одно очко.

При одинаковом количестве очков у двух и более команд, места в турнирной таблице определяются следующим
образом:

• по результатам всех личных встреч между соперничающими командами;

• по наибольшему количеству выигранных боёв во всех матчах онлайн-части;

• при равенстве всех этих показателей, в случае взятия призового места, все спорящие команды получают приз
согласно занятому месту либо переходят в стадию плей-офф.

Организатор оставляет за собой право добавить игровой сервер в случае необходимости.
